Minigames

A continuously growing collection of recreations of games from the first generation of videogames onward — written in Rust, playable right here in your browser.

Every Game ships two takes. The Faithful preserves the original's rules, difficulty and feel, wrapped in a modern shell. The Remix grows out of that Faithful once it ships. A Game is Done only when both are live — and only a Done Game unlocks the next.

Games

Pong · 1972 · Done

The one that started it — and the Collection's first Done Game. The Faithful keeps the original's paddle-angle deflection and rally speed-up; PULSE, its Remix, reimagines the same core with spin, power shots, pickups and three modes. Pick either on the opening screen.
